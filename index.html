hi@leekinney.is

I am a front-end web developer living in Philadelphia, PA

I like to create websites that are user friendly and implement the latest web standards. I'm currently working towards learning AngularJS, becoming an expert in web accessibility and contributing to more open source projects. I am a teacher, TA and events coordinator for Girl Develop It Philadelphia, organizer for LadyHacks and the Philly Women in Tech Summit. Sometimes I can be found blogging at leekinney.is, SheTechPHL and Philly Hikes. In my spare time I love to hike, bike, camp and travel as much as possible.
